    • GHOST-CIPA-ROOT
      2.1. GHOST Context Integration and Prompt Assembly Scope GHOST-CIPA-ROOT This file is a draft parent requirement file in the GHOST StrictDoc requirement tree.

      It defines ownership, scope, and placeholder headings for Context Integration and Prompt Assembly.

      Detailed SuperPrompt projection, context validation, conflict handling, Hard Rule integration, and finalization requirements shall be owned by the corresponding child files.

      This file shall not duplicate detailed requirements owned by child or sibling files.
      • GHOST-SRS-FILE-TREE
        2.8.1. Requirement File Tree Parent Anchor GHOST-SRS-FILE-TREE This requirement is the parent anchor for the approved StrictDoc child-file tree. Connected child dummy files shall reference this UID, or a child-file parent UID, through StrictDoc parent references so generated HTML can display navigable requirement relationships.
    GHOST-SPPB-ROOT

    This file shall not duplicate detailed requirements owned by sibling files.
