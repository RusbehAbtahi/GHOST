        GHOST-HARD-RULES-ROOT
        Hard Rule Integration and Finalization Scope
        This file shall define:

        - current runtime and GUI facts related to Hard Rules;
        - the distinction between per-agent static instructions and the planned global Hard Rules capability;
        - the current A5 and final-send boundaries; and
        - the approved human-controlled and inspectable future direction.
        REQUIREMENT
