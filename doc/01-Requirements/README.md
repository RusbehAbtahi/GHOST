
# GHOST Requirements

This folder contains the requirements documentation for **GHOST**.

## Current status

The requirements are currently being refactored from the legacy Markdown format into **StrictDoc**.

This work is still **in progress**.

## Legacy requirements

The previous complete requirement documents are stored in:

```text
doc/01-Requirements/legacy_md/
````

These files are kept as historical/source material during the migration.

## StrictDoc refactoring

The new StrictDoc-based requirement files are stored in:

```text
doc/01-Requirements/strictdoc/
```

The root SRS is:

```text
doc/01-Requirements/strictdoc/00_ghost_srs.sdoc
```
